#  Resume–Job Description Skill Matcher

An NLP-based system that calculates how well a resume matches a job description using **sentence embeddings and cosine similarity**.

---

##  Features
- Semantic resume–JD matching
- Match percentage score
- Clean Streamlit UI
- Recruiter-focused use case

---

##  Technology Stack
- Python
- NLP
- Sentence Transformers
- Streamlit
- Scikit-learn
